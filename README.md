# MONINTION — System Monitor

Dashboard de monitoramento de sistema em tempo real. Roda em Windows, Linux (Kali, Ubuntu) e Raspberry Pi.

## Instalação

```bash
pip install -r requirements.txt
python server.py
```

Acesse: `http://localhost:5050`
